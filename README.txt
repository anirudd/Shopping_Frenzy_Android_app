Shopping Frenzy Android App

This is an android 2.3+ app that can be used to search for products under a given product category around the world. It has localisation support for many languages. The results offered by the app can be filtered according to price, brand or both. 

The app uses Google Shopping API to search for products. 

The app has a seamless user interface and covers a wide variety of products.

The app was made using Eclipse software and official Android SDK provided by Google.

The app requires connection to internet to display the results.

The app can be downloaded at :
